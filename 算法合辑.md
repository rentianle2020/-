### 约瑟夫环

n个人（从1~n排号）每次枪毙第k个，问最终剩下的是第几号人。

**举例**

n = 5, k = 3

0,1,2,3,4 -> 0,1,3,4 -> **3,4,0,1 -> 0,1,2,3**

删除下标为k - 1的元素后，第k个元素变为下标为0的元素。所以它从k变为0，倒推，(index + k) % n
